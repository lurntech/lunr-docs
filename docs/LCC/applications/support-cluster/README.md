# Support Cluster

The Support Cluster includes applications used to remotely diagnose and troubleshoot device related issues. The applications include:&#x20;

* **Network Logger:** tool collecting network activity logs from devices. &#x20;
* **Action Logger:** tool collecting device activity logs.
* **Sensor Logger:** tool collecting sensor logs from devices.
* **Debug Bridge:** tool for establishing remote ADB connection to devices.
* **Mission Control:** tool for gaining full control over devices remotely.&#x20;
